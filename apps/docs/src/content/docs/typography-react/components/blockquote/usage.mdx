Blockquote displays quoted text with optional citation attribution. Supports default, accent, and filled variants.

## Basic Usage

<ComponentPreview
  htmlCode={`<blockquote class="pm-blockquote pm-blockquote--default pm-blockquote--md">
  Life is what happens when you are busy making other plans.
</blockquote>`}
  reactCode={`import { Blockquote } from "@paramanu/typography-react"

export default function App() {
  return (
    <Blockquote cite="John Lennon">
      Life is what happens when you are busy making other plans.
    </Blockquote>
  )
}`}
/>
