Heading renders semantic h1-h6 elements with independent visual sizing. The level maps to a default size but can be overridden.

## Basic Usage

<ComponentPreview
  htmlCode={`<h2 class="pm-heading pm-heading--size-2xl pm-heading--weight-bold">Section Title</h2>`}
  reactCode={`import { Heading } from "@paramanu/typography-react"

export default function App() {
  return <Heading level={2}>Section Title</Heading>
}`}
/>
