## API

### SkipNav

| Prop       | Type     | Default          | Description                        |
| ---------- | -------- | ---------------- | ---------------------------------- |
| `targetId` | `string` | `"main-content"` | The `id` of the element to skip to |

### SkipNavTarget

| Prop | Type     | Default          | Description                      |
| ---- | -------- | ---------------- | -------------------------------- |
| `id` | `string` | `"main-content"` | The `id` the skip link points to |
