## Accessibility

- Uses `role="progressbar"` with `aria-valuenow`, `aria-valuemin`, `aria-valuemax`
- Value expressed as 0-100 for screen readers

## Best Practices

- Use for page-level navigation transitions
- Use `useNProgress` hook for automatic trickle behavior
- Call `start()` on navigation begin and `done()` on completion
