## Accessibility

- Uses `aria-hidden` to toggle visibility for assistive technology
- Content inside the overlay should contain a loading indicator or text

## Best Practices

- Use on positioned containers (`position: relative`)
- Toggle `visible` prop instead of mounting/unmounting for animations
- Use blur effect sparingly to maintain performance
