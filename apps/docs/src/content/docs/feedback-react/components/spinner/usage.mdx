Spinner displays an animated loading indicator with a visually hidden "Loading" label for accessibility.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-spinner pm-spinner--md pm-spinner--primary" role="status">
  <span class="sr-only">Loading</span>
</div>`}
  reactCode={`import { Spinner } from "@paramanu/feedback-react"

export default function App() {
  return <Spinner />
}`}
/>
