Progress displays linear progress with support for determinate/indeterminate modes, striped patterns, and labels.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-progress pm-progress--md pm-progress--primary" role="progressbar" aria-valuenow="60" aria-valuemin="0" aria-valuemax="100">
  <div class="pm-progress__track">
    <div class="pm-progress__fill" style="width: 60%;"></div>
  </div>
</div>`}
  reactCode={`import { Progress } from "@paramanu/feedback-react"

export default function App() {
  return <Progress value={60} />
}`}
/>
