## API

| Prop       | Type                    | Default  | Description                         |
| ---------- | ----------------------- | -------- | ----------------------------------- |
| `target`   | `string \| HTMLElement` | `"body"` | Target element or selector          |
| `disabled` | `boolean`               | `false`  | Renders in-place without portalling |
