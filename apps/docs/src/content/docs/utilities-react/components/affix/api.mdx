## API

| Prop       | Type                | Default | Description                   |
| ---------- | ------------------- | ------- | ----------------------------- |
| `position` | `"top" \| "bottom"` | `"top"` | Which edge to stick to        |
| `offset`   | `SpacingScale`      | -       | Distance from the sticky edge |
