## Accessibility

- Uses `role="list"` with `aria-label="Progress"`.
- Active step uses `aria-current="step"`.
- Step indicators are `aria-hidden="true"` (decorative).

## Best Practices

- Show 3-7 steps for optimal usability.
- Use vertical orientation for longer step sequences.
