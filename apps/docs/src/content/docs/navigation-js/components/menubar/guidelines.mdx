## Accessibility

| Key               | Action                |
| ----------------- | --------------------- |
| `ArrowRight`      | Move to next item     |
| `ArrowLeft`       | Move to previous item |
| `Enter` / `Space` | Activate focused item |
| `Home`            | Move to first item    |
| `End`             | Move to last item     |

- Uses `role="menubar"` on the container and `role="menuitem"` on items.
