## Accessibility

| Key         | Action                |
| ----------- | --------------------- |
| `ArrowDown` | Move to next item     |
| `ArrowUp`   | Move to previous item |
| `Enter`     | Activate focused item |
| `Escape`    | Close menu            |

- Uses `role="menu"` on the container.
- Triggered via right-click (contextmenu event).
