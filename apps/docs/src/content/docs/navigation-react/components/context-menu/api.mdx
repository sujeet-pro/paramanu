## CtxMenu Props

| Prop   | Type                   | Default | Description      |
| ------ | ---------------------- | ------- | ---------------- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"`  | Size preset      |
| `open` | `boolean`              | `false` | Visibility state |

## CSS Classes

| Class               | Description  |
| ------------------- | ------------ |
| `pm-ctx-menu`       | Root element |
| `pm-ctx-menu--open` | Open state   |
