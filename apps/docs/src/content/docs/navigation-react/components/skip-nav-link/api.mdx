## SkipLink Props

| Prop       | Type        | Default                  | Description       |
| ---------- | ----------- | ------------------------ | ----------------- |
| `href`     | `string`    | `"#main-content"`        | Target element ID |
| `children` | `ReactNode` | `"Skip to main content"` | Link text         |

## CSS Classes

| Class          | Description                                      |
| -------------- | ------------------------------------------------ |
| `pm-skip-link` | Root element (visually hidden, visible on focus) |
