## CSS Classes

| Class                 | Purpose        |
| --------------------- | -------------- |
| `pm-text`             | Base           |
| `pm-text--size-{s}`   | Font size      |
| `pm-text--weight-{w}` | Font weight    |
| `pm-text--color-{c}`  | Color          |
| `pm-text--truncate`   | Truncation     |
| `pm-text--italic`     | Italic         |
| `pm-text--inline`     | Inline display |
