## CSS Classes

| Class                  | Purpose        |
| ---------------------- | -------------- |
| `pm-list`              | Base           |
| `pm-list--ordered`     | Ordered list   |
| `pm-list--unordered`   | Unordered list |
| `pm-list--spacing-{s}` | Spacing        |
| `pm-list--style-{t}`   | Marker style   |
| `pm-list--unstyled`    | Remove markers |
