## Accessibility

- Use proper `<thead>`, `<tbody>`, `<th>` elements for screen reader navigation.
- Add `<caption>` for table descriptions.

## Best Practices

- Use `striped` variant for tables with many rows to improve readability.
- Enable `hoverable` for interactive tables with row selection.
