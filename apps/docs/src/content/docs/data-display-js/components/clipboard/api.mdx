## Props

| Prop      | Type                   | Default | Description                        |
| --------- | ---------------------- | ------- | ---------------------------------- |
| `size`    | `"sm" \| "md" \| "lg"` | `"md"`  | Padding and icon size              |
| `value`   | `string`               | -       | Text to copy                       |
| `timeout` | `number`               | `2000`  | Duration of the copied state in ms |
