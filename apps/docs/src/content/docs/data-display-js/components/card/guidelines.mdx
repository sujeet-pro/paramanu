## Accessibility

- Use `<article>` semantics when the card represents a standalone piece of content.
- Interactive cards should be focusable and respond to keyboard activation.
- Ensure sufficient color contrast between card content and background.

## Best Practices

- Use `elevated` for primary content and `outline` for secondary groupings.
- Keep card content concise and scannable.
- Use the `interactive` prop only when the entire card is clickable.
