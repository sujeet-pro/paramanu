## Accessibility

- Use descriptive text content for screen reader context.
- Pair with nearby text that describes what the badge represents.

## Best Practices

- Use `pill` for numeric counters (e.g., notification counts).
- Use semantic colors: `success` for positive, `danger` for errors.
