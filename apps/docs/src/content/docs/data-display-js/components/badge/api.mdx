## CSS Classes

| Class                 | Description      |
| --------------------- | ---------------- |
| `pm-badge`            | Base badge class |
| `pm-badge--{variant}` | Variant modifier |
| `pm-badge--{size}`    | Size modifier    |
| `pm-badge--{color}`   | Color modifier   |
| `pm-badge--pill`      | Pill shape       |
