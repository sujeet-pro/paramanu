## Props

| Prop         | Type           | Default | Description                                   |
| ------------ | -------------- | ------- | --------------------------------------------- |
| `size`       | `"sm" \| "md"` | `"md"`  | Cell padding and font size                    |
| `selectable` | `boolean`      | `false` | Enables row selection with hover/focus states |
| `bordered`   | `boolean`      | `false` | Adds borders around cells                     |
