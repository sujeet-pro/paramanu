## API

| Prop        | Type                 | Default    | Description                       |
| ----------- | -------------------- | ---------- | --------------------------------- |
| `placement` | `HovercardPlacement` | `"bottom"` | Position relative to the trigger  |
| `open`      | `boolean`            | `false`    | Whether the hover card is visible |
