A compact dismiss button with a close icon, intended for use inside alerts, dialogs, toasts, and banners.

## Basic Usage

<ComponentPreview
  htmlCode={`<button class="pm-close-btn pm-close-btn--md" aria-label="Close">
  <svg aria-hidden="true" width="1em" height="1em" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2.5"><line x1="18" y1="6" x2="6" y2="18"/><line x1="6" y1="6" x2="18" y2="18"/></svg>
</button>`}
  reactCode={`import { CloseBtn } from "@paramanu/buttons-react"

export default function App() {
  return <CloseBtn onClick={() => console.log("closed")} />
}`}
/>
