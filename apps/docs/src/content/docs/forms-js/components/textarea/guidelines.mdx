## Accessibility

| Feature         | Detail                              |
| --------------- | ----------------------------------- |
| Keyboard        | Standard textarea keyboard behavior |
| `aria-invalid`  | Applied when `invalid` is true      |
| `aria-disabled` | Applied when `disabled` is true     |
| Focus ring      | Visible via `:focus-visible`        |

## Best Practices

- Always pair with a visible `<label>` or `aria-label`.
- Use `resize="none"` only when the layout requires fixed dimensions.
- Set appropriate `rows` for expected content length.
