The Checkbox component provides a toggle input with support for checked, unchecked, and indeterminate states.

## Basic Usage

<ComponentPreview
  htmlCode={`<label class="pm-checkbox pm-checkbox--md">\n  <input type="checkbox" class="pm-checkbox__input" />\n  <span class="pm-checkbox__indicator">&#10003;</span>\n  <span class="pm-checkbox__label">Accept terms</span>\n</label>`}
  reactCode={`import { Checkbox } from "@paramanu/forms-react"\n\nexport default function App() {\n  return <Checkbox>Accept terms</Checkbox>\n}`}
/>
