## Accessibility

| Feature                    | Detail                             |
| -------------------------- | ---------------------------------- |
| `role="combobox"`          | Input role                         |
| `aria-autocomplete="list"` | Autocomplete behavior              |
| `aria-expanded`            | Reflects open state                |
| Keyboard                   | Arrow keys navigate, Enter selects |
