The Mentions component provides a text area that triggers a suggestion dropdown when the user types a mention trigger character like `@`.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-mentions pm-mentions--outline pm-mentions--md">\n  <textarea class="pm-input pm-input--outline pm-input--md" placeholder="Type @ to mention someone..." rows="3"></textarea>\n</div>`}
  reactCode={`import { Mentions } from "@paramanu/forms-react"\n\nexport default function App() {\n  return (\n    <Mentions>\n      <textarea placeholder="Type @ to mention someone..." rows={3} />\n    </Mentions>\n  )\n}`}
/>
