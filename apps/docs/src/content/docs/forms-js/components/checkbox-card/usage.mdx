The Checkbox Card renders a checkbox inside a card layout, providing a more visual selection experience than a standard checkbox.

## Basic Usage

<ComponentPreview
  htmlCode={`<label class="pm-chk-card pm-chk-card--md">\n  <input type="checkbox" class="pm-chk-card__input" />\n  <div class="pm-chk-card__content">Option A</div>\n</label>`}
  reactCode={`import { ChkCard } from "@paramanu/forms-react"\n\nexport default function App() {\n  return <ChkCard>Option A</ChkCard>\n}`}
/>
