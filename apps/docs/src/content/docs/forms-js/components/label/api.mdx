## Props

| Prop       | Type                           | Default | Description              |
| ---------- | ------------------------------ | ------- | ------------------------ |
| `size`     | `"xs" \| "sm" \| "md" \| "lg"` | `"md"`  | Label size               |
| `disabled` | `boolean`                      | `false` | Disabled state           |
| `required` | `boolean`                      | `false` | Shows required indicator |
| `htmlFor`  | `string`                       | —       | Associated input ID      |
