The Input component provides a styled text input field with multiple visual variants, sizes, and states for form data entry.

## Basic Usage

<ComponentPreview
  htmlCode={`<input class="pm-input pm-input--outline pm-input--md" type="text" placeholder="Enter text..." />`}
  reactCode={`import { Input } from "@paramanu/forms-react"\n\nexport default function App() {\n  return <Input placeholder="Enter text..." />\n}`}
/>
