## Overview

CommandPalette provides a keyboard-driven interface for searching and executing commands, similar to VS Code's command palette or Spotlight search. Typically opened with a hotkey.

## Usage

<Tabs>
  <TabItem label="HTML">
    ```html
    <div class="pm-command-palette">
      <input class="pm-command-palette__input" role="combobox"
        aria-expanded="true" aria-autocomplete="list"
        placeholder="Type a command..." />
      <div class="pm-command-palette__list" role="listbox">
        <div class="pm-command-palette__group" role="group">
          <div class="pm-command-palette__item" role="option">New File</div>
          <div class="pm-command-palette__item pm-command-palette__item--active"
            role="option" aria-selected="true">Save</div>

      </div>

    ```
</TabItem>
  <TabItem label="React">
    ```tsx

      CommandPaletteList, CommandPaletteItem,
    } from "@paramanu/overlays-react"

    <CommandPalette open={isOpen} onClose={handleClose}>
    ```

## Guidelines

- Open with a recognizable hotkey (Cmd+K or Ctrl+K)
- Support keyboard navigation (arrow keys, Enter, Escape)
- Show an empty state when no results match
- Group related commands together

## API

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | `false` | Whether the palette is visible |
| `onClose` | `() => void` | - | Callback when the palette should close |
