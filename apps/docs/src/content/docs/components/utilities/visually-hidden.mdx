import { Tabs, TabItem } from "@astrojs/starlight/components"

## Overview

VisuallyHidden removes content from the visual flow while keeping it accessible to assistive technologies. Uses the clip-rect technique recommended by WebAIM.

## Usage

<Tabs>
  <TabItem label="HTML">
    ```html
    <button>
      <svg aria-hidden="true">...</svg>
      <span class="pm-visually-hidden">Close dialog</span>
    </button>
    ```
</TabItem>
  <TabItem label="React">
    ```tsx
    import { VisuallyHidden } from "@paramanu/utilities-react"

    <button>
      <Icon aria-hidden />
      <VisuallyHidden>Close dialog</VisuallyHidden>
    </button>
    ```
</TabItem>
</Tabs>

## Guidelines

- Use for icon-only buttons that need accessible labels
- Use `focusable` for skip navigation links
- Do not use to hide decorative content (use `aria-hidden` instead)

## API

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `focusable` | `boolean` | `false` | Element becomes visible when focused |
| `as` | `ElementType` | `"span"` | The HTML element to render |
