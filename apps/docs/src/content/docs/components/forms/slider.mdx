## Accessibility

| Feature | Detail |
|---|---|
| `role="slider"` | Slider role on root |
| `aria-valuemin/max` | Range boundaries |
| `aria-orientation` | Horizontal or vertical |
| Keyboard | Arrow keys adjust value |

</TabItem>
