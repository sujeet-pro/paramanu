The Native Select wraps the browser's native `<select>` element with consistent styling and a custom dropdown arrow.

<Tabs syncKey="component-tab">
  <TabItem label="Usage">

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-native-sel pm-native-sel--outline pm-native-sel--md">\n  <select class="pm-native-sel__field">\n    <option value="">Select...</option>\n    <option value="apple">Apple</option>\n    <option value="banana">Banana</option>\n  </select>\n  <span class="pm-native-sel__arrow" aria-hidden="true">&#9662;</span>\n</div>`}
  reactCode={`import { NativeSel } from "@paramanu/forms-react"\n\nexport default function App() {\n  return (\n    <NativeSel>\n      <option value="">Select...</option>\n      <option value="apple">Apple</option>\n      <option value="banana">Banana</option>\n    </NativeSel>\n  )\n}`}
/>

## Accessibility

| Feature | Detail |
|---|---|
| Native `<select>` | Full browser accessibility support |
| Arrow indicator | Decorative, `aria-hidden="true"` |
| Keyboard | Standard select keyboard behavior |

</TabItem>
  <TabItem label="API">

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `"outline" \| "filled" \| "unstyled"` | `"outline"` | Visual variant |
| `size` | `"xs" \| "sm" \| "md" \| "lg"` | `"md"` | Select size |
| `invalid` | `boolean` | `false` | Invalid state |
| `disabled` | `boolean` | `false` | Disabled state |
| `fullWidth` | `boolean` | `false` | Full-width layout |

## CSS Classes

| Class | Description |
|---|---|
| `pm-native-sel` | Wrapper base class |
| `pm-native-sel__field` | Native select element |
| `pm-native-sel__arrow` | Custom arrow indicator |
