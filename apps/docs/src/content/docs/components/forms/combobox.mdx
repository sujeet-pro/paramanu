## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-combobox pm-combobox--outline pm-combobox--md">\n  <input class="pm-combobox__input" role="combobox" aria-expanded="false" aria-autocomplete="list" aria-haspopup="listbox" placeholder="Search..." />\n  <div class="pm-combobox__listbox" role="listbox">\n    <div role="option">Apple</div>\n    <div role="option">Banana</div>\n  </div>\n</div>`}
  reactCode={`import { Combobox } from "@paramanu/forms-react"\n\nexport default function App() {\n  return (\n    <Combobox placeholder="Search...">\n      <div role="option">Apple</div>\n      <div role="option">Banana</div>\n    </Combobox>\n  )\n}`}

## Accessibility

| Feature                    | Detail                             |
| -------------------------- | ---------------------------------- |
| `role="combobox"`          | Input role                         |
| `aria-autocomplete="list"` | Autocomplete behavior              |
| `aria-expanded`            | Reflects open state                |
| Keyboard                   | Arrow keys navigate, Enter selects |

</TabItem>

## Props

| Prop        | Type                                  | Default     | Description       |
| ----------- | ------------------------------------- | ----------- | ----------------- |
| `variant`   | `"outline" \| "filled" \| "unstyled"` | `"outline"` | Visual variant    |
| `size`      | `"xs" \| "sm" \| "md" \| "lg"`        | `"md"`      | Size              |
| `invalid`   | `boolean`                             | `false`     | Invalid state     |
| `disabled`  | `boolean`                             | `false`     | Disabled state    |
| `open`      | `boolean`                             | `false`     | Open state        |
| `fullWidth` | `boolean`                             | `false`     | Full-width layout |

## CSS Classes

| Class                  | Description       |
| ---------------------- | ----------------- |
| `pm-combobox`          | Base class        |
| `pm-combobox__input`   | Text input        |
| `pm-combobox__listbox` | Options container |
