## Accessibility

| Feature         | Detail                            |
| --------------- | --------------------------------- |
| `role="button"` | Activatable drop target           |
| Keyboard        | Enter/Space activates file picker |

| Prop       | Type      | Default | Description            |
| ---------- | --------- | ------- | ---------------------- |
| `disabled` | `boolean` | `false` | Disabled state         |
| `dragging` | `boolean` | `false` | Active drag-over state |
