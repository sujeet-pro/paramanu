  htmlCode={`<div class="pm-date-picker pm-date-picker--outline pm-date-picker--md">\n  <input type="text" class="pm-date-picker__input" role="combobox" aria-expanded="false" aria-haspopup="dialog" placeholder="Select date..." />\n  <span class="pm-date-picker__icon" aria-hidden="true">&#128197;</span>\n</div>`}

## Accessibility

| Feature | Detail |
|---|---|
| `role="combobox"` | Input role |
| `aria-haspopup="dialog"` | Opens a calendar dialog |
| Keyboard | Enter opens calendar, Escape closes |

</TabItem>

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `"outline" \| "filled" \| "unstyled"` | `"outline"` | Visual variant |
| `size` | `"xs" \| "sm" \| "md" \| "lg"` | `"md"` | Size |
| `invalid` | `boolean` | `false` | Invalid state |
| `disabled` | `boolean` | `false` | Disabled state |
| `open` | `boolean` | `false` | Open state |

## CSS Classes

| Class | Description |
|---|---|
| `pm-date-picker` | Base class |
| `pm-date-picker__input` | Text input |
| `pm-date-picker__icon` | Calendar icon |
| `pm-date-picker__popover` | Calendar popover |
