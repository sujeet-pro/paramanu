## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `layout` | `"vertical" \| "horizontal" \| "inline"` | `"vertical"` | Form layout |
| `gap` | `"sm" \| "md" \| "lg"` | `"md"` | Spacing between controls |
