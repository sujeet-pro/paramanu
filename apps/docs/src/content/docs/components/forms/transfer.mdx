The Transfer component provides a dual-list interface for moving items between source and target panels.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-transfer pm-transfer--md" role="group" aria-label="Transfer">...</div>`}

## Accessibility

| Feature        | Detail                                    |
| -------------- | ----------------------------------------- |
| `role="group"` | Groups the two panels                     |
| Keyboard       | Tab navigates between panels and controls |

</TabItem>

## Props

| Prop       | Type                           | Default | Description    |
| ---------- | ------------------------------ | ------- | -------------- |
| `size`     | `"xs" \| "sm" \| "md" \| "lg"` | `"md"`  | Transfer size  |
| `disabled` | `boolean`                      | `false` | Disabled state |
