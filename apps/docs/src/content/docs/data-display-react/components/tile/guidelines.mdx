## Accessibility

- Tiles render as `<button>` elements for keyboard accessibility.
- Use `aria-selected` to indicate the selected state.
- Disabled tiles use both `disabled` and `aria-disabled`.

## Best Practices

- Use tiles for selection patterns where users pick one or more options.
- Group tiles visually in a grid layout for clear scanning.
