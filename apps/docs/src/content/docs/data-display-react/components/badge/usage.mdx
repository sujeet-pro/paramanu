Badge is a small inline label used to display status, counts, or categories. Supports filled, outline, and subtle variants with multiple color palettes.

## Basic Usage

<ComponentPreview
  htmlCode={`<span class="pm-badge pm-badge--filled pm-badge--md pm-badge--primary">New</span>`}
  reactCode={`import { Badge } from "@paramanu/data-display-react"

export default function App() {
  return <Badge>New</Badge>
}`}
/>
