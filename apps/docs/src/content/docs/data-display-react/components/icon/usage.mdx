Icon is a container for SVG icons with consistent sizing, color inheritance, and optional spin animation. Supports accessible and decorative modes.

## Basic Usage

<ComponentPreview
  htmlCode={`<span class="pm-icon pm-icon--md pm-icon--inherit" role="img" aria-label="Star">
  <svg viewBox="0 0 24 24" fill="currentColor"><circle cx="12" cy="12" r="10" /></svg>
</span>`}
  reactCode={`import { Icon } from "@paramanu/data-display-react"

export default function App() {
  return (
    <Icon label="Star">
      <svg viewBox="0 0 24 24" fill="currentColor"><circle cx="12" cy="12" r="10" /></svg>
    </Icon>
  )
}`}
/>
