Divider renders a horizontal or vertical separator line. It supports solid, dashed, and dotted styles, and can include a label.

## Basic Usage

<ComponentPreview
  htmlCode={`<hr class="pm-divider pm-divider--horizontal pm-divider--solid" />`}
  reactCode={`import { Divider } from "@paramanu/primitives-react"

export default function App() {
  return <Divider />
}`}
/>

## With Label

<ComponentPreview
  htmlCode={`<div class="pm-divider pm-divider--horizontal pm-divider--solid pm-divider--with-label" role="separator">OR</div>`}
  reactCode={`import { Divider } from "@paramanu/primitives-react"

export default function App() {
  return <Divider label="OR" />
}`}
/>
