Wrap lays out children in a wrapping row with consistent gap. Useful for tag lists, chip groups, and responsive layouts.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-wrap pm-wrap--gap-3">
  <span style="background: #e2e8f0; padding: 4px 12px; border-radius: 4px;">Tag 1</span>
  <span style="background: #e2e8f0; padding: 4px 12px; border-radius: 4px;">Tag 2</span>
  <span style="background: #e2e8f0; padding: 4px 12px; border-radius: 4px;">Tag 3</span>
</div>`}
  reactCode={`import { Wrap } from "@paramanu/primitives-react"

export default function App() {
  return (
    <Wrap gap="3">
      <span>Tag 1</span>
      <span>Tag 2</span>
      <span>Tag 3</span>
    </Wrap>
  )
}`}
/>
