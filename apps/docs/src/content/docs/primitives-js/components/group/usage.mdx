Group arranges related elements together, supporting attached borders (for button groups), grow, and wrap.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-group pm-group--horizontal pm-group--gap-3" role="group">
  <button type="button">Action 1</button>
  <button type="button">Action 2</button>
  <button type="button">Action 3</button>
</div>`}
  reactCode={`import { Group } from "@paramanu/primitives-react"

export default function App() {
  return (
    <Group gap="3">
      <button>Action 1</button>
      <button>Action 2</button>
      <button>Action 3</button>
    </Group>
  )
}`}
/>
