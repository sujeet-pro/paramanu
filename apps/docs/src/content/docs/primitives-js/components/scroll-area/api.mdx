| Class                         | Purpose           |
| ----------------------------- | ----------------- |
| `pm-scroll`                   | Base              |
| `pm-scroll--vertical`         | Vertical scroll   |
| `pm-scroll--horizontal`       | Horizontal scroll |
| `pm-scroll--bordered`         | Border            |
| `pm-scroll--scrollbar-hidden` | Hide scrollbar    |
