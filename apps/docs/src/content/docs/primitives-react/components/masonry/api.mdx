## React Props

| Prop      | Type                    | Default | Description       |
| --------- | ----------------------- | ------- | ----------------- |
| `columns` | `2 \| 3 \| 4 \| 5 \| 6` | `3`     | Number of columns |
| `gap`     | `SpacingScale`          | --      | Gap between items |

## CSS Classes

| Class                  | Purpose      |
| ---------------------- | ------------ |
| `pm-masonry`           | Container    |
| `pm-masonry--cols-{n}` | Column count |
| `pm-masonry--gap-{n}`  | Gap size     |
