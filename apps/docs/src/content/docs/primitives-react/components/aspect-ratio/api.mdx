## React Props

| Prop    | Type                                                                      | Default  | Description  |
| ------- | ------------------------------------------------------------------------- | -------- | ------------ |
| `ratio` | `"1/1" \| "2/3" \| "3/2" \| "4/3" \| "3/4" \| "16/9" \| "9/16" \| "21/9"` | `"16/9"` | Aspect ratio |

## CSS Classes

| Class                | Purpose                |
| -------------------- | ---------------------- |
| `pm-aspect`          | Base                   |
| `pm-aspect--{ratio}` | Ratio (e.g., `--16-9`) |
