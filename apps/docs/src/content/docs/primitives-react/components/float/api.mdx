## React Props

| Prop        | Type             | Default     | Description        |
| ----------- | ---------------- | ----------- | ------------------ |
| `placement` | `FloatPlacement` | `"top-end"` | Position           |
| `offset`    | `"0"-"6"`        | --          | Distance from edge |

## CSS Classes

| Class                   | Purpose  |
| ----------------------- | -------- |
| `pm-float`              | Base     |
| `pm-float--{placement}` | Position |
| `pm-float--offset-{n}`  | Offset   |
