The Textarea component provides a multi-line text input with configurable resize behavior, variants, and sizes.

## Basic Usage

<ComponentPreview
  htmlCode={`<textarea class="pm-textarea pm-textarea--outline pm-textarea--md pm-textarea--resize-vertical" rows="3" placeholder="Enter text..."></textarea>`}
  reactCode={`import { Textarea } from "@paramanu/forms-react"\n\nexport default function App() {\n  return <Textarea placeholder="Enter text..." rows={3} />\n}`}
/>
