The Number Input component provides a numeric text field with stepper buttons for incrementing and decrementing values.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-num-input pm-num-input--outline pm-num-input--md">\n  <input class="pm-input pm-input--outline pm-input--md" type="number" inputmode="numeric" />\n  <div class="pm-num-input__stepper">\n    <button type="button" class="pm-num-input__increment" aria-label="Increment">&#9650;</button>\n    <button type="button" class="pm-num-input__decrement" aria-label="Decrement">&#9660;</button>\n  </div>\n</div>`}
  reactCode={`import { NumInput } from "@paramanu/forms-react"\n\nexport default function App() {\n  return <NumInput />\n}`}
/>
