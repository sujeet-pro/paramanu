The Segmented Control provides a set of mutually exclusive options in a button-group layout.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-seg-ctrl pm-seg-ctrl--md" role="radiogroup">\n  <button type="button" class="pm-seg-ctrl__item" aria-pressed="true">Day</button>\n  <button type="button" class="pm-seg-ctrl__item">Week</button>\n  <button type="button" class="pm-seg-ctrl__item">Month</button>\n</div>`}
  reactCode={`import { SegCtrl } from "@paramanu/forms-react"\n\nexport default function App() {\n  return (\n    <SegCtrl>\n      <button aria-pressed="true">Day</button>\n      <button>Week</button>\n      <button>Month</button>\n    </SegCtrl>\n  )\n}`}
/>
