## Props

| Prop        | Type                           | Default | Description       |
| ----------- | ------------------------------ | ------- | ----------------- |
| `size`      | `"xs" \| "sm" \| "md" \| "lg"` | `"md"`  | Control size      |
| `fullWidth` | `boolean`                      | `false` | Full-width layout |

## CSS Classes

| Class                     | Description         |
| ------------------------- | ------------------- |
| `pm-seg-ctrl`             | Base class          |
| `pm-seg-ctrl__item`       | Segment button      |
| `pm-seg-ctrl--full-width` | Full-width modifier |
