The Editable Text component displays text that can be clicked to switch into edit mode, allowing inline editing.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-editable pm-editable--md" role="group">\n  <span>Click to edit</span>\n</div>`}
