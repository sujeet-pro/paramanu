## Accessibility

| Feature                   | Detail                                 |
| ------------------------- | -------------------------------------- |
| `role="combobox"`         | Trigger button role                    |
| `aria-expanded`           | Reflects open state                    |
| `aria-haspopup="listbox"` | Indicates dropdown type                |
| Keyboard                  | Enter/Space opens, Arrow keys navigate |
