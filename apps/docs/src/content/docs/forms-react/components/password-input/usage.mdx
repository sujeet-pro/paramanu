The Password Input wraps a standard input with a visibility toggle, allowing users to reveal or conceal their password.

## Basic Usage

<ComponentPreview
  htmlCode={`<div class="pm-pwd-input pm-pwd-input--outline pm-pwd-input--md">\n  <input class="pm-input pm-input--outline pm-input--md" type="password" placeholder="Enter password..." />\n  <button type="button" class="pm-pwd-input__toggle" aria-label="Show password">Show</button>\n</div>`}
  reactCode={`import { PwdInput } from "@paramanu/forms-react"\n\nexport default function App() {\n  return <PwdInput placeholder="Enter password..." />\n}`}
/>
