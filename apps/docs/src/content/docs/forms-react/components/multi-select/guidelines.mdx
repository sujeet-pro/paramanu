## Accessibility

| Feature                | Detail                              |
| ---------------------- | ----------------------------------- |
| `aria-multiselectable` | Listbox supports multiple selection |
| `role="combobox"`      | Trigger role                        |
| Keyboard               | Space/Enter toggles selection       |
