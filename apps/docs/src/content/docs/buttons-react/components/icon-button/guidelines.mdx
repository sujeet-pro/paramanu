## Accessibility

- Requires `aria-label` since there is no visible text
- Uses semantic `<button>` element
- Loading state sets `aria-busy="true"`

## Best Practices

- Always provide a descriptive `aria-label`
- Use circle shape for floating or prominent actions
- Pair with tooltips for discoverability
